# SPACE INVADERS 🛸

A retro Space Invaders clone in pure HTML5 Canvas.

## How to Play

- **← → or A/D** — Move ship
- **SPACE** — Shoot
- Destroy all aliens before they reach you!

## Features

- 80s arcade aesthetic with CRT scanline effects
- 3 lives
- Increasing difficulty per level
- Score tracking
- Retro color scheme (green, red, cyan, yellow)

## Run

Simply open `index.html` in any browser.

https://fritzkebot.github.io/invaders
